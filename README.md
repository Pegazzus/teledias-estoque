# Sistema de Estoque - Teledias Telecom

Sistema web para controle de estoque de rádios de comunicação.

## Tecnologias

- Node.js + Express
- SQLite (better-sqlite3)
- HTML/CSS/JavaScript
- Chart.js

## Como Executar Localmente

```bash
npm install
npm start
```

Acesse: http://localhost:3000
